## Booking Strategy — How to Avoid Double-Booking

**The rule:** Book one site per night. Only move to a backup when the primary is confirmed unavailable.

### Decision tree for each night

```
PRIMARY available? → Book it → Done. Do NOT also book a backup.
PRIMARY sold out (❌)? → Book BACKUP 1 immediately → Update status below.
BACKUP 1 also sold out? → Book BACKUP 2 → Update status below.
All reservable options gone? → Plan for FCFS or dispersed → Note below.
```

### FCFS nights (special case)

A few nights are **FCFS-only primaries** (Joshua Tree, North Campground Bryce). For these, you may optionally pre-book a reservable backup as insurance:

> **If you pre-book a backup for an FCFS night, set a calendar reminder to cancel it at least 3 days before the night.** Most recreation.gov and ReserveCalifornia sites allow penalty-free cancellation up to 2–3 days out. If you arrive and get the FCFS spot: cancel the backup that same day.
